一。PL的角色职责及追求
1.带领团队做好目标管理及项目管理，确保业务按时，高质，高效地持续成功交付，达成业务目标
2.以身作则，执行关键业务活动，交付极致产品，承担技术把关指导，架构看护优化及技术能力储备等职责，构建团队技术核心竞争力。
3.结合业务实施绩效管理，分解团队目标，关注和辅导下属，建设有血性，有灵魂的高绩效精兵队伍，实现团队和个人的共同成长。
